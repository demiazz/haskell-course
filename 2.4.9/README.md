## Основы программирования - Стандартные классы типов

### Задание

Напишите функцию с сигнатурой:

```
avg :: Int -> Int -> Int -> Double
```

вычисляющую среднее значение переданных в нее аргументов:

```
GHCi> avg 3 4 8
5.0
```

### Исходный код

```
avg :: Int -> Int -> Int -> Double
avg = undefined
```
